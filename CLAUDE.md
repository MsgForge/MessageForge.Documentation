# MessageForge.Documentation

> Project documentation, architecture decisions, handoff notes, and prototypes.

## Structure

| Path            | Role                                  |
| --------------- | ------------------------------------- |
| `architecture/` | Architecture Decision Records (ADRs)  |
| `handoff/`      | Technical handoff docs from PoC phase |
| `prototypes/`   | HTML prototypes and UI mockups        |

## Conventions

- Markdown for all text documents
- No code here — code lives in its respective subsystem
- Handoff docs from PoC are reference-only; do not treat as current spec
